To use the hacks, go to the specified game and in the url bar, type 'javascript:' (without the quotes) and copy/paste the code after it.

Pong: document.onkeydown = checkKey; function checkKey(e) { if (e.keyCode == '72') { onePlayerScore += 1; } else if (e.keyCode == '74') { nextRally() } }

Press 'h' to increase your score, press 'j' to skip the questions
